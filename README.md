A C-based steganography project that hides secret text messages inside image files using the Least Significant Bit (LSB) technique. The application allows users to securely embed and extract hidden data without visibly altering the original image.

🔹 Features

Encode secret messages into BMP images

Decode hidden messages from images

Uses LSB technique for data embedding

Maintains image quality after encoding

Command-line interface for easy usage

🛠️ Technologies Used

C Programming

File Handling

Bitwise Operations

Binary Data Manipulation
